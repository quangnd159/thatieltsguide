{{< spoiler text="🌱 Structure" >}}
- (number) suitable verb
{{< /spoiler >}}

{{< alert icon="📝" >}}
**NOTE:** This is a very natural way to talk about numbers.
{{< /alert >}}

**Examples**

_In 2004, about 3,700 burglaries happened._

_About 25% of families had a refrigerator in 1998._
